# Phishing URL Detection System

Проект создан в рамках дипломной работы и предназначен для выявления фишинговых URL с помощью методов машинного обучения.

## Основные возможности
- Анализ и классификация URL-адресов
- Извлечение лексических признаков
- Использование моделей машинного обучения:
  - Logistic Regression
  - Random Forest
  - XGBoost
- REST API на Flask для предсказаний

## Результаты
- Точность модели: ~98%

## Датасеты

Для обучения и тестирования модели используется датасет фишинговых и безопасных URL-адресов.  

**Источник:**  
Датасет взят из открытого репозитория на GitHub: [ESDAUNG/PhishDataset](https://github.com/ESDAUNG/PhishDataset/blob/main/data_imbal%20-%2055000.xls).  

**Описание:**  
Файл `data_imbal - 55000.xls` содержит примеры URL, разделенные на фишинговые и безопасные, и предназначен для анализа и построения моделей обнаружения фишинга.
